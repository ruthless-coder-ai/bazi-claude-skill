# 分类解答手册（用户问什么，看哪里，怎么答）

前提：已完成 analysis.md 的前三步（强弱、格局、喜忌）。每类问题按"看点 → 组织回答"执行。
每类回答都遵守两区结构与推理链外露（analysis.md 第六步）、判词→镜子的翻译（voice.md）。
用户没有指定问题时，**不要默认倒全景**——走 SKILL.md 的问题闸门，问用户想先看哪块。

## 1. 总论 / 性格 /「我的命怎么样」

**看点**：日主意象（fundamentals.md 天干表）→ 月令季节底色 → 全局最强的 1–2 个十神 → 格局主题 → 喜忌。
**回答结构**：
1. 一段话人物画像（日主+最强十神组合，讲性格光谱：亮面+暗面各一）
2. 格局给的人生主题（适合走什么路）
3. 全盘的"药"（喜什么忌什么，对应的方向建议：行业、环境）
4. 一句整体大运走势预览（哪个年龄段进入喜用神运）

## 2. 人生阶段（早年 / 青年 / 中年 / 晚年）

**看点**：宫位柱（年→早年、月→青年、日→中年、时→晚年）的喜忌 + 该年龄段所行大运的喜忌，两层叠加。
**回答结构**：按阶段分段，每段讲：
- 该柱坐喜用还是忌神（底色：顺 or 磨）
- 覆盖该年龄段的大运是升是降（具体到虚岁区间和公历年份）
- 该阶段的主题事件倾向（宫位十神：如月柱伤官→青年期靠才华闯、易换赛道）
时辰未知时，晚年部分明确说明只能看大运、精度有限。

## 3. 婚姻感情

**看点**：
- **配偶星**：男命看财星（正财=正缘），女命看官星（正官=正缘，七杀=强势缘分/情感波折）。星的状态：透干藏支？逢生逢克？有几个（多则缘分杂）？无则缘分应期靠大运流年引动。
- **配偶宫**（日支）：坐喜用→配偶得力；坐忌神→相处耗心；被合→配偶宫有牵动；被冲刑→婚姻易有波折期。
- **桃花**：位置与数量（详见 fundamentals.md 神煞节）。
- **女命特别看**：伤官旺而官星弱（易挑剔冲突）；**男命特别看**：比劫旺而财星弱（竞争、聚少）。
- **婚期/恋爱应期**：大运流年配偶星透出、或流年合/冲配偶宫之年。
**回答结构**：配偶画像（星+宫的五行十神意象）→ 相处模式与磨合点 → 未来 3–5 年感情机会/波动窗口（给具体年份）。
已婚用户跳过"婚期"，讲相处与波动窗口；不知对方八字时不做合婚断言。
婚变类凶象（宫逢冲、星受克）何时说怎么说，按 voice.md 的三分支决策程序。

## 4. 事业 / 财运

**看点**：
- **官杀**＝职位、平台、管理权 → 旺而为喜：适合体制/大厂/走职级；
- **食伤**＝才华、作品、专业输出 → 旺而为喜：适合专业路线、创作、自由职业；
- **财星**＝钱本身。正财=薪资积累，偏财=生意投资。**食伤生财**链通=能把才华变现；**比劫夺财**=合伙谨慎、财来财去；
- **印星**＝证书学历资源靠山 → 旺：适合学术、稳定机构；
- 行业五行建议：从喜用神五行推（木=教育文创植物；火=能源文娱互联网；土=地产农业中介；金=金融机械司法；水=贸易物流传媒流动性行业）。只作参考方向，不武断。
**回答结构**：适合的赛道类型（打工/体制/创业/专业自由职业，从格局+十神组合推）→ 财富模式（挣钱靠什么、漏财在哪）→ 事业上升窗口（喜用神大运区间+近几年流年）。
用户问跳槽/创业等具体决策：看当年流年喜忌+是否冲驿马/冲禄，给"窗口评估"而非"该不该"。

## 5. 家庭六亲 / 子女

**看点**：
- **父母**：印星（母）、偏财（父）状态 + 年月柱喜忌。年月为喜→出身助力多；逢冲刑→早年家庭变动。
- **兄弟朋友**：比劫状态。比劫为喜→朋友是助力；为忌→防合伙破财。
- **子女**：子女星（男命看官杀，女命看食伤）+ 子女宫（时柱）。星宫皆喜→子女缘厚且得力；星弱宫空（逢空亡）→缘分偏淡或聚少离多，不说"无子"。
- 时辰未知 → 子女话题明确降级："只能从子女星看大致缘分，宫位信息缺失。"
**回答分寸**：六亲话题容易触伤（父母离异、失亲），语气收敛，先描述"缘分深浅、助力方向"，不主动断生死伤病。

## 6. 健康

**看点**：五行对应脏腑：木=肝胆/筋；火=心/血液/眼；土=脾胃/消化；金=肺/呼吸道/皮肤；水=肾/泌尿/骨。
- 某行**过旺**或**过弱（0 个）**→ 对应系统是薄弱面
- 忌神所在五行被流年再加强、或本命关键支被冲 → 该年注意对应部位
**回答分寸**：只讲"注意保养的方向"+生活建议，不诊断、不预言疾病，建议体检交给医生。健康凶象说不说、怎么说，走 voice.md 的三分支决策程序。

## 7. 近期运势（今年 / 明年 / 未来 N 年 / 某个时段）

**看点**（层层缩小）：
1. **当前大运**：十年底色（喜/忌，主题十神）
2. **目标流年**：干支五行对照喜忌；流年十神主题（见下表）；流年与原局的合冲刑害触发什么宫位
3. 用户问到半年/季度/月份 → 用排盘 JSON 里的**流月**表：标出顺利月份 2–3 个、需谨慎月份 1–2 个

流年十神速查（当年主旋律）：
| 流年十神 | 当年主题 |
|---|---|
| 比肩/劫财 | 合作与竞争、朋友往来多、注意分财 |
| 食神/伤官 | 表达输出、新想法、跳出舒适区；女命感情注意沟通 |
| 正财/偏财 | 求财应酬忙碌；男命感情机会 |
| 正官/七杀 | 职位责任压力、名誉考核；女命感情机会 |
| 正印/偏印 | 学习证书文书、贵人长辈、休整充电 |

**回答结构**：底色（大运）一句 → 当年 2–3 条具体倾向（事业/感情/财务/健康中挑相关的）→ 时段节奏（流月）→ 一条可执行建议。
超过当前大运范围的年份（如用户问 10 年后），说明需按彼时大运另看，简要带过即可。

## 8. 通用规则

- **先答所问**：用户问什么先直接回答什么，答完再问"还想看哪块"——默认不倒全景（闸门规则见 SKILL.md 4b）。
- **复用已排的盘**：同一会话内追问不重新跑脚本（除非换人、改生日）。
- **给年份要具体**：说"2027 丁未年"而不是"过两年"。
- **不用万能免责句**：诚实做在每一条推断里（推理链外露、置信度分层、可被经历证伪——analysis.md 第六/七步），而不是结尾一句话盖住全文。"传统诠释框架而非科学预测"在首次介绍体系时说明一次即可。
