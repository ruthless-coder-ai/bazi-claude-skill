# 八字分析方法论（拿到排盘 JSON 后，按此顺序推理）

分析主线是传统子平法：**日主强弱 → 格局 → 用神喜忌 → 宫位 → 大运流年**。
每一步先在心里推完，再决定怎么讲给用户；"怎么讲"的规则在 voice.md，动笔前必读。

## 第一步：定日主强弱

日主（日干）是"我"。强弱决定整个盘的喜忌方向。三看：

1. **得令**（权重最高）：日主五行在月令处于什么状态？
   - 当令（月令与日主同五行或生日主）→ 旺相
   - 简表：木旺于春（寅卯辰）、火旺于夏（巳午未）、金旺于秋（申酉戌）、水旺于冬（亥子丑）、土旺于四季末（辰未戌丑月）
2. **得地**：日主在四个地支的藏干里有没有同五行的根（通根）。有本气根最强（如甲见寅），余气根弱。
3. **得势**：天干地支中生我（印）、同我（比劫）的字多不多。

三者得二即偏强，三者皆失即偏弱。介于中间的说"中和偏强/偏弱"即可，不必强判。

> 注意：不是强就好、弱就差。强弱只决定"需要什么"，盘的好坏看用神是否有力、组合是否流通。

## 第二步：取格局（定人生主题）

以**月令本气藏干**（或月令藏干中透出天干者）对日主的十神定格。如月令藏干的十神是正官→正官格。

| 格局 | 人生主题（一句话） |
|---|---|
| 正官格 | 走规则内的路：体制、大机构、名誉驱动 |
| 七杀格 | 在压力和竞争中立身，宜武职/销售/创业攻坚 |
| 正财格 | 务实积累型，勤恳理财，稳中求富 |
| 偏财格 | 经营流通型，善抓机会，财来财去规模大 |
| 正印格 | 学术文教型，靠名声资历吃饭 |
| 偏印格 | 专业技术/偏门领域，深度胜过广度 |
| 食神格 | 才艺表达型，靠作品和口碑，生活家 |
| 伤官格 | 才华锋芒型，创新颠覆，不宜困于科层 |
| 建禄/羊刃格（月令即日主本气/刃） | 白手起家型，靠自身实力，不靠祖荫 |

比肩/劫财不单独成格（归入建禄羊刃论）。特殊格局（从格、化格：全局一边倒、日主无根无助时弃命而从）只需点到：若五行统计极端（某行 0 个且日主无根），提示"可能入从格，喜忌反转"，并按两种可能分别简述。

## 第三步：定用神喜忌（全盘的药）

- **扶抑为主**：日主强 → 喜克泄耗（官杀、食伤、财），忌印比再帮；日主弱 → 喜生扶（印、比劫），忌财官食伤再耗。
- **调候为辅**：极寒（冬月）盘喜火，极燥（夏月）盘喜水，即使与扶抑冲突也要兼顾——先看命局有没有调候字，没有则大运补上时明显转好。
- **通关**：两行相战势均力敌时，取中间通关之行为用（如金木交战用水）。

输出时把喜忌落到**具体五行**（如"喜水木，忌土金"），后面所有大运流年吉凶、方位颜色行业建议都从这里推。

## 第四步：宫位系统（谁、何时、何处）

| 柱 | 六亲宫位 | 人生阶段（大致） |
|---|---|---|
| 年柱 | 祖上、父母宫（与月柱共看） | 早年 1–16 岁 |
| 月柱 | 父母、兄弟姐妹宫 | 青年 17–32 岁 |
| 日柱 | 自己（干）与配偶宫（支） | 中年 33–48 岁 |
| 时柱 | 子女宫、下属晚辈 | 晚年 49 岁以后 |

宫位 × 十神 × 喜忌三层叠加读：
- 宫位里坐的是**喜用神** → 该宫位对应的人/阶段给力；坐**忌神** → 该处多耗损阻力。
- 宫位逢冲刑 → 该宫位的人事易有变动分离。
- 星与宫分看：如"夫星"（正官）在年柱而配偶宫（日支）坐忌神，可解读为"缘分早现但相处磨合多"。

## 第五步：大运流年推法（何时应事）

层级：**大运定十年底色 → 流年定当年之事 → 流月定月份节奏**。

1. **大运吉凶**：大运干支五行对照喜忌。喜用神运 → 顺境窗口；忌神运 → 蓄力期。干管前五年、支管后五年（也有整体看的，取干支综合、支略重）。
2. **应事扳机**：大运/流年干支与**原局**发生的合冲刑害是事件的触发器：
   - 冲/合**配偶宫**（日支）→ 感情婚姻动（结婚、分手、聚少离多，吉凶看喜忌）
   - 流年干支为**配偶星**且成势 → 婚恋机会年
   - 冲**驿马**或驿马逢值 → 搬迁、换工作、出国
   - 流年与日柱**天克地冲**（干克干+支冲支）→ 动荡较大的年份，宜守不宜攻
   - 流年见**七杀**且为忌 → 压力、竞争、健康注意；见**印**化杀 → 有惊无险
3. **岁运并临**（大运流年同干支）：能量加倍，吉更吉凶更凶，需重点提示。
4. **流月**：以流年定基调后，流月干支五行对照喜忌，标出全年 2–3 个最顺月份与 1–2 个需谨慎月份即可，不逐月啰嗦。

## 第六步：输出结构与确定性分层（必须遵守）

命理输出最大的诚实问题：用户分不清哪句是**算出来的**、哪句是**你说的**。解决办法是把分层做进输出结构，让用户在每一句话上都知道自己读到的是什么。

**两区结构**（每次解读都用）：

1. **「盘面」区**——只放脚本可追溯的事实：四柱干支、藏干十神、五行统计、神煞、大运流年干支、脚本的临界提醒。平铺直述，一个判断都不带。这一区用户可以拿去任何排盘工具复核。
2. **「解读」区**——从这里开始是诠释。**每个判断都要露出推理链**："盘面依据 → 传统读法 → 所以倾向……"（例："你日主己土生在卯月、四柱只有年支一个火根【盘面】→ 传统上算失令少助，取印比为用【框架内的标准读法】→ 所以逢火土的大运流年通常更顺【推断】"）。让用户看得见每一步推导，有不同意的地方可以指着具体一环反驳。

**三级置信度**，在解读的行文里自然带出（不是机械贴标签）：

- **计算事实**：脚本输出的一切。说法："你的盘是……"
- **强推断**：子平框架内没有分歧的标准读法（日主强弱、喜忌方向、宫位含义）。说法："按传统的读法……""这个盘的标准判法是……"
- **一家之言**：流派有分歧、或证据链较长的推论（具体应期、神煞细节、性格细节）。说法："往细里说就见仁见智了，我的读法是……""这一条你可以拿自己的经历检验。"

**不要用结尾的万能免责句**（"以上仅供参考"盖住全文，等于哪句都不负责）。取而代之：每条推断本身逐条可辩驳——推理链露在外面，置信度标在句子里，并主动邀请用户拿真实经历对账（见第七步）。诠释框架本身是传统文化而非科学，这一点在**首次介绍这套体系时**说一次即可，不必每次重复。

其余分寸规则（详细的执行规则、正反例在 voice.md，动笔前必读）：
- 焦虑中的用户（问病、问离婚、问失业）：先接住情绪，再给盘面，结尾落到当下能做的事。
- 重大决定（离婚、辞职、大额投资）：盘面信息 + 利弊，决定权明确还给用户。
- 医疗、法律、投资的具体决策：提醒找专业人士。
- 凶信息何时说、怎么说：按 voice.md 的三分支决策程序，不许可的字眼也在那里。

## 第七步：历史校准（把宣判变成协作验证）

解读不该是单向宣判。邀请用户给出 **2–3 件过去的大事和年份**（换工作、结婚/分手、搬迁、大病、发财或破财），拿这套读法去对账——先证明读法在这个盘上灵不灵，再往前推。

**对账方法**：把每件事的年份换成当年的大运+流年干支，检查是否存在对应的触发机制——
- 该年冲/合了哪个宫位？（冲配偶宫之年恰好分手？）
- 该年透出的十神是否对应事件类型？（伤官年跳槽顶撞领导、财年进账、官杀年升职或高压）
- 是否喜忌转换点？（换大运那年人生换挡）
- 驿马逢冲之年是否搬迁？

**诚实汇报，三种结果都如实说**：
- 对上了：指出具体机制（"你 2019 年离职，恰是己亥年伤官透干、又冲你的驿马——这类变动应期在你盘上是灵的"），并说明这**提高了**同类断语后续的置信度。
- 对不上：明说（"2021 年你说的低谷，盘面上看不出明显机制，这条我对不上"），**降低**相关断语的置信度，不要硬圆。硬圆一次，前面建立的全部诚信清零。
- 部分对上：说到什么程度就是什么程度。

**校准的副产品——定盘**：时辰临界、晚子时这类"两可盘"（脚本提醒里会标出），正确用法就是排出两个盘各自对一遍过去，哪个对得多用哪个。这是传统命理本来就有的"定盘"手续，明说无妨。

**前推时引用校准结果**："你盘上'驿马逢冲则动'验证过两次，所以 2028 年那次冲，我给的置信度比一般盘高。"——校准过的机制敢说得笃定些，没校准过的保持一家之言的口吻。
