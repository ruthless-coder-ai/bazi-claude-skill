# 八字分析方法论（拿到排盘 JSON 后，按此顺序推理）

分析主线是传统子平法：**日主强弱 → 格局 → 用神喜忌 → 宫位 → 大运流年**。
每一步先在心里推完，再决定怎么讲给用户。

## 第一步：定日主强弱

日主（日干）是"我"。强弱决定整个盘的喜忌方向。三看：

1. **得令**（权重最高）：日主五行在月令处于什么状态？
   - 当令（月令与日主同五行或生日主）→ 旺相
   - 简表：木旺于春（寅卯辰）、火旺于夏（巳午未）、金旺于秋（申酉戌）、水旺于冬（亥子丑）、土旺于四季末（辰未戌丑月）
2. **得地**：日主在四个地支的藏干里有没有同五行的根（通根）。有本气根最强（如甲见寅），余气根弱。
3. **得势**：天干地支中生我（印）、同我（比劫）的字多不多。

三者得二即偏强，三者皆失即偏弱。介于中间的说"中和偏强/偏弱"即可，不必强判。

> 注意：不是强就好、弱就差。强弱只决定"需要什么"，盘的好坏看用神是否有力、组合是否流通。

## 第二步：取格局（定人生主题）

以**月令本气藏干**（或月令藏干中透出天干者）对日主的十神定格。如月令藏干的十神是正官→正官格。

| 格局 | 人生主题（一句话） |
|---|---|
| 正官格 | 走规则内的路：体制、大机构、名誉驱动 |
| 七杀格 | 在压力和竞争中立身，宜武职/销售/创业攻坚 |
| 正财格 | 务实积累型，勤恳理财，稳中求富 |
| 偏财格 | 经营流通型，善抓机会，财来财去规模大 |
| 正印格 | 学术文教型，靠名声资历吃饭 |
| 偏印格 | 专业技术/偏门领域，深度胜过广度 |
| 食神格 | 才艺表达型，靠作品和口碑，生活家 |
| 伤官格 | 才华锋芒型，创新颠覆，不宜困于科层 |
| 建禄/羊刃格（月令即日主本气/刃） | 白手起家型，靠自身实力，不靠祖荫 |

比肩/劫财不单独成格（归入建禄羊刃论）。特殊格局（从格、化格：全局一边倒、日主无根无助时弃命而从）只需点到：若五行统计极端（某行 0 个且日主无根），提示"可能入从格，喜忌反转"，并按两种可能分别简述。

## 第三步：定用神喜忌（全盘的药）

- **扶抑为主**：日主强 → 喜克泄耗（官杀、食伤、财），忌印比再帮；日主弱 → 喜生扶（印、比劫），忌财官食伤再耗。
- **调候为辅**：极寒（冬月）盘喜火，极燥（夏月）盘喜水，即使与扶抑冲突也要兼顾——先看命局有没有调候字，没有则大运补上时明显转好。
- **通关**：两行相战势均力敌时，取中间通关之行为用（如金木交战用水）。

输出时把喜忌落到**具体五行**（如"喜水木，忌土金"），后面所有大运流年吉凶、方位颜色行业建议都从这里推。

## 第四步：宫位系统（谁、何时、何处）

| 柱 | 六亲宫位 | 人生阶段（大致） |
|---|---|---|
| 年柱 | 祖上、父母宫（与月柱共看） | 早年 1–16 岁 |
| 月柱 | 父母、兄弟姐妹宫 | 青年 17–32 岁 |
| 日柱 | 自己（干）与配偶宫（支） | 中年 33–48 岁 |
| 时柱 | 子女宫、下属晚辈 | 晚年 49 岁以后 |

宫位 × 十神 × 喜忌三层叠加读：
- 宫位里坐的是**喜用神** → 该宫位对应的人/阶段给力；坐**忌神** → 该处多耗损阻力。
- 宫位逢冲刑 → 该宫位的人事易有变动分离。
- 星与宫分看：如"夫星"（正官）在年柱而配偶宫（日支）坐忌神，可解读为"缘分早现但相处磨合多"。

## 第五步：大运流年推法（何时应事）

层级：**大运定十年底色 → 流年定当年之事 → 流月定月份节奏**。

1. **大运吉凶**：大运干支五行对照喜忌。喜用神运 → 顺境窗口；忌神运 → 蓄力期。干管前五年、支管后五年（也有整体看的，取干支综合、支略重）。
2. **应事扳机**：大运/流年干支与**原局**发生的合冲刑害是事件的触发器：
   - 冲/合**配偶宫**（日支）→ 感情婚姻动（结婚、分手、聚少离多，吉凶看喜忌）
   - 流年干支为**配偶星**且成势 → 婚恋机会年
   - 冲**驿马**或驿马逢值 → 搬迁、换工作、出国
   - 流年与日柱**天克地冲**（干克干+支冲支）→ 动荡较大的年份，宜守不宜攻
   - 流年见**七杀**且为忌 → 压力、竞争、健康注意；见**印**化杀 → 有惊无险
3. **岁运并临**（大运流年同干支）：能量加倍，吉更吉凶更凶，需重点提示。
4. **流月**：以流年定基调后，流月干支五行对照喜忌，标出全年 2–3 个最顺月份与 1–2 个需谨慎月份即可，不逐月啰嗦。

## 第六步：断语分寸（必须遵守）

- 说**倾向和概率**，不说定数："这段时间事业容易有变动"，而非"你明年必然离职"。
- **凶不吓人**：健康、婚变、破财类信息给出提醒 + 可行建议（如"某年注意肠胃、避免高风险投资"），不用恐吓性字眼，不预言死亡重病。
- **吉不打包票**：机会窗口讲"适合主动争取"，不承诺结果。
- 用户处于焦虑中（问病、问离婚、问失业）时，先回应情绪，再给盘面信息，结尾落到当下能做的事。
- 不替用户做重大决定（离婚、辞职、大额投资），盘面信息 + 利弊分析，决定权还给用户。
- 涉及具体医疗、法律、投资决策时提醒咨询专业人士。
